دليل سريع - كيفية التعديل والنشر
ملفات الموقع موجودة في هذا المجلد:
- index.html
- styles.css
- script.js
- assets/logo.png

أهم التعديلات التي قد تريدين عملها:
1) تغيير رقم الواتساب:
   افتحي file index.html وابحثي عن 'YOUR_NUMBER' واستبدليه برقمك بصيغة دولية بدون زائد.
   مثال: 9665XXXXXXXX

2) تغيير البريد الإلكتروني الافتراضي:
   في index.html وابحثي عن 'your-email@example.com' واستبدليه ببريدك.

3) رفع صور الأعمال الحقيقية:
   ضعي الصور داخل مجلد assets وحدثي عناصر <img> في قسم المشاريع.

نشر على GitHub Pages:
- أنشئي مستودع جديد، ارفعي الملفات ضمن الفرع main.
- اذهبي إلى Settings > Pages واختر branch main / root ثم احفظي.
- سيتم نشر الموقع تلقائيًا.

إذا أردتي، أستطيع تحويل هذا القالب لاحقًا إلى WordPress أو أضيف نموذج اتصال يرسل بيانات مباشرة إلى بريدك عبر خدمة Formspree أو Google Forms.
